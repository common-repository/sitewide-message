== Description ==

Simply. Display a message at the top of every page on your website. You can change the background colour, text colour and URL location.

= New Features =
* Scheduled Message - Choose when to show or hide a message on your website.
* Link Location - Open link url in new window or same window.

Activate the plugin. Nevigate to the settings menu under Dashboard, and away you go.

= New Features =

Request new features in the support forms.

== Installation ==

1. Upload and extract the .zip file downloaded to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start using the plugin
